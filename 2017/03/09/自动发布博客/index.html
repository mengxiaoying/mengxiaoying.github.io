使用 Hexo 静态博客生成器搭建自己的个人博客

Mar 9, 2017 | Hits
Inhalte

官方网站：https://hexo.io/zh-cn/

# 全局安装 hexo
npm install -g hexo-cli

# 初始化一个博客
hexo init blog

# 切入 blog 目录
cd blog

# 启动预览
hexo server

# 创建博客
hexo new 文章标题

结合 Github Pages 自动发布博客

一：Github 创建域名仓库

二：修改博客中的 _config.yml 文件中

deploy:
  type: git
  repo: https://heima05:abc123456@github.com/lipengzhou/lipengzhou.github.io.git

三：安装插件

npm install hexo-deployer-git --save

四：发布

hexo generate --deploy
hexo deploy --generate
